# Reddalyzer
### 

#### Technologies (Mostly AWS)

* Infrastructure as Code: `CloudFormation` and `Terraform`
* Compute: `EC2`
* Streaming: `Kinesis Firehose`
* Data Lake: `S3`
* Data discovery: `Glue`
* Data Analytics:`Athena`
* Data Visualization and Business Intelligence: `QuickSight`
